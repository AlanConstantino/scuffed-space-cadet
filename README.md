# scuffed-space-cadet
A scuffed 2D shoot 'em up.

## Installation
1. Download this repository and unzip it.
2. Download pygame `pip3 install pygame`
3. Run game.py `python3 game.py`

## Controls
- ESC or P: Pause game
- ArrowKeys: Movement
- H: Show/hide HUD
- M: Mute game
- R: Restart game

## Features
- Sound
- Boss
- Music
- Score
